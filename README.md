# React course. Matheus Battisti - Hora de Codar
Course to learn React starter. Understand its characteristics before NextJS.

```bash
Using {
  Website:
    - React
    - TypeScript (interfaces + tsx)
    - Dotenv

  Api and DB:
    - JSON Server
}
```

## How to start the application

### Run front-end
```bash
npm start

```

### Run back-end
```bash
npm run backend

```
